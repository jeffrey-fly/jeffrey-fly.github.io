How to use addr2line

Posted on 2026-03-01 In debug

addr2line 是 GNU Binutils 提供的一个调试工具，用于将程序地址转换为源代码文件 + 行号。
其底层依赖 ELF 文件中的 DWARF 调试信息。

典型使用场景：

  • core dump 分析
  • 崩溃栈回溯解析
  • 日志中只有地址信息
  • 线上二进制与调试符号分离
